JAVASCRIPT

    Javascript es un lenguaje utilizado por principiantes, personas que esten iniciando en el mundo de diseño web, como tambien para profesionales.
    Este lenguaje no requiere ser compilado ya que esta util herramienta funciona del lado del cliente, los encargados de interpretar este codigo son los navegadores. Muchos tienden a confundir Java con JavaScript no existe ni la mas minima relación ya que son dos lenguajes completamente diferentes.
    Aqui una de las mas grandes diferencias entre JAVA y JAVASCRIPT, y es que Javasript puede ser incorporado en cualquier pagina web, puede ser ejecutado sin la necesidad de la instalacion de un programa como tal para ser visualizado
    Podemos decir que Javascript es un lenguaje interpretado, mientras que Java es Orientado a Objetos.
