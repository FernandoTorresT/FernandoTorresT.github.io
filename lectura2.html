Cascade Style Sheets

    Si deseamos aprender a diseñar sitios web, CSS3 es una de las herramientas mas importantes en cuanto a diseño, junto a HTML y JAVASCRIPT se crea un excelente conjunto para llevar a cabo este proposito de darle un aspecto atractivo y unico al sitio web. CSS significa Cascade Style Sheets, tambien llamado en español Hojas de Estilo en Cascada, basicamente funciona en conjunto con archivos HTML. Entonces para crear un sitio web debes tener conocimiento y tener la capacitan de manipular tanto HTML como CSS. Unas de las ventajas y mayores logros utilizados y creados por CSS3 es que nos ofrece el servicio y la posibilidad de crear y añadir animaciones y efectos. Tambien gracias a las transiciones nos permite el cambio de la apariencia y el comportamiento de un elemento cada vez que se da un cambio de estado. CSS3 ademas de permitirme el cambio de color en mis elemento, tambien me permite crear grandientes, en sus versiones anteriores esto no era posible y tenias que incluir imagenes pero ahora en esta version puedes hacerlo desde el mismo archivo CSS.
