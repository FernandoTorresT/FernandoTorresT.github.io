Cascade Style Sheets

    Si deseamos aprender a diseñar sitios web, CSS3 es una de las herramientas mas importantes en cuanto a diseño, junto a HTML y JAVASCRIPT se crea un excelente conjunto para llevar a cabo este proposito de darle un aspecto atractivo y unico al sitio web. CSS significa Cascade Style Sheets, tambien llamado en español Hojas de Estilo en Cascada, basicamente funciona en conjunto con archivos HTML. Entonces para crear un sitio web debes tener conocimiento y tener la capacitan de manipular tanto HTML como CSS.

En este ejemplo se observan tres nuevas etiquetas, <title> que se encuentran dentro del head del documento que sirve para darle el título a la página en el navegador y el <h1> ubicado en el body del documento que define el encabezado y la etiqueta <p> que define un párrafo. Atributos de etiquetas

Los atributos sirven para dar ciertas características a las etiquetas, como cambiar el color de fondo o texto; de hecho ciertos casos los tags no fun- cionan por si solos, deben llevar un atributo de apoyo para ejecutarse de manera apropiada, esto se define en la etiqueta de apertura.
