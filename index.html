Curso de diseño web - UNAD

En esta pagina web encontrara un curso que le serviran para aprender los fundamentos sobre tecnologias web como lo es CSS, HTML, JAVASCRIPT. El curso fue diseñado por estudiantes de la universidad como producto del curso de diseño de sitios web.

Objetivos del curso


Objetivo general

Desarrollar la maquetación y diseño de una página web, teniendo en cuenta una estructura, y secciones que compone una página web, aplicando una correcta planeación, estableciendo una vista previa y organización de la pagina

  • Identificar las herramientas para la maquetación y diseño de la página web
  • Planear una correcta estructura y distribución de las secciones en la página teniendo en cuenta que sea cómoda e intuitiva para el usuario.
  • Implementar y tener en cuenta la regulación del diseño en la pagina para que esta sea agradable para usuario
