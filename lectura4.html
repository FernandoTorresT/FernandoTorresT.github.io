Diseño Web

    El diseño web es un conjunto de actividades que consta de diseño, planificación, implementación, y mantenimientos de sitios web. Esta tecnologia abarca aspectos importantes a la hora de crear un sitio web como lo es el diseño de interfaz, el diseño grafico web, y podemos mencionar las experiencia de usuario.
    Normalmente equipos de trabajadores se encargan de cubrir diferentes aspectos del proceso del diseño, aunque tambien existen algunos diseñadores independientes que elaboran sitios web solos.
