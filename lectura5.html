¿Que es GitHub

    Esta herramienta no es muy conocida para muchos desarrolladores principiantes e incluso para desarrolladores profesionales y es que sin duda es una herramienta fundamental para la hora de programar, y se trata de una herramienta que se encarga de controlar versiones, mas especificamente GIT que es un servicio publico de GitHub.
    Es muy importante a la hora de trabajar en equipo ya que nos puede evitar muchos dolores de cabeza, nos permite mantener el codigo ordenadamente, y lo mas importante tener versiones por si se presenta errores.
    Sabemos que tener copias de seguridad es una de las medidas mas importantes a la hora de crear un sitio web ya que poseemos de toda la informacion, por ejemplo, Fotografias, ficheros, codigos de programacion ETC.
    Comunmente los medios utilizados para guardar estos documentos son Dropbox o Drive, pero en caso de esta herramientas tenemos algo mucho mejor llamado:Repositorios Git. Es la creación de varias versiones que nos permiten remediar errores si se han cometido y desplegar la que creamos que fue desarrollada correctamente, o nos permiten trabajar funcionalidades nuevas sin necesidad de editar la version funcional y asi no afectar el proyecto.
